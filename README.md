# Seating Chart Maker

**Live Demo:** https://zeejfps.github.io/seating_chart_maker/

A web-based seating chart tool for managing event table assignments. Features drag-and-drop guest placement, CSV guest list import, an interactive floor plan view, and undo/redo support.

Built with Svelte 5, TypeScript, and Vite.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
